Dancher Silvanie

Software Developer in opleiding

Hallo! Ik ben Dancher, een gedreven software developer in mijn laatste jaar aan het Techniek College Rotterdam op de Prins Alexanderlaan. Ik heb een sterke passie voor het bouwen van webapplicaties en het oplossen van technische uitdagingen.

Tijdens mijn opleiding heb ik gewerkt met moderne webtechnologieën zoals HTML, CSS, PHP met object-georiënteerd programmeren en MySQL-databases. Ik vind het leuk om ideeën om te zetten in werkende applicaties — van een persoonlijk mediabeheer systeem tot een volledig functionele voetbalsite.

Naast programmeren ben ik een grote voetballiefhebber, game ik graag in mijn vrije tijd, luister ik muziek en kijk ik series en films. Deze interesses inspireren mij ook in mijn projectkeuzes — zoals mijn FotMob-geïnspireerde voetbalsite.
