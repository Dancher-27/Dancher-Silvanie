# Dancher-Silvanie
Portfolio Website — Dancher Silvanie

Dit is mijn persoonlijke portfolio website, volledig vanaf nul gebouwd met HTML, CSS en JavaScript.
De website laat mijn vaardigheden, projecten en achtergrond als software developer zien.

Er zijn geen frameworks of libraries gebruikt — alle code is handmatig geschreven.

Pagina’s
Pagina	Beschrijving
index.html	Homepagina met introductie en typed-effect
about.html	Informatie over mij met foto’s
skills.html	Overzicht van mijn technische vaardigheden
projects.html	Portfolio met mijn projecten
hobbies.html	Persoonlijke interesses
contact.html	Contactformulier en contactgegevens
Functionaliteiten

Typed effect
Dynamisch tekst-effect op de homepage.

Scroll animaties
Elementen verschijnen wanneer ze in beeld komen met de IntersectionObserver API.

Geanimeerde skill bars
Vaardigheden worden visueel weergegeven met progress bars.

Foto carousel
Swipebare galerij met navigatie op de “About” pagina.

Responsive navigatie
Hamburger menu voor mobiele schermen.

Contactformulier
Formulier met client-side validatie.

Gebruikte technologieën

HTML5 — semantische structuur

CSS3 — Flexbox, Grid en animaties

JavaScript — interactiviteit en animaties

Er zijn geen externe frameworks of libraries gebruikt.

Design

De website gebruikt een modern dark theme design.

Primaire kleur: Indigo #6C63FF

Accentkleur: Teal #00D9A3

Layout: Flexbox en CSS Grid

Responsive: geschikt voor desktop, tablet en mobiel

Installatie

De website heeft geen installatie nodig.

Open een van de HTML-bestanden direct in de browser of via een lokale server zoals XAMPP.

Voorbeeld:

http://localhost/project-portfolio/
